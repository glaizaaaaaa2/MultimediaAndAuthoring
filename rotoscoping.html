Process

Step 1

Prepare the Applications

Set up Adobe After Effects and Photoshop. Organize your project files and folders on your desktop before starting.

Step 2

Split the Video into 5-Second Clips

Cut your reference footage into manageable 5-second segments so each week covers one clip.

Step 3

After Effects — Export via Render Queue

Import clips into After Effects. Go to Composition → Add to Render Queue, set format to PNG Sequence, and render.

Step 4

Import Rendered Frames into Photoshop

Open Photoshop → File → Open → select first frame → check "Image Sequence" → set FPS → open.

Step 5

Edit the Rendered Frames

Trace and color each frame in Photoshop using brush tool, paint bucket, and layer masks.
